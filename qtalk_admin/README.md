## docker image

[docker image 说明](https://hub.docker.com/_/php/)，该项目使用的是php:5.6-apache.

## 操作说明

```
进入container
sudo docker exec -it php /bin/bash

停止该服务
sudo docker stop php

启动该服务
sudo docker start php

重启该服务
sudo docker restart php

删除该container
sudo docker rm -f php
```
