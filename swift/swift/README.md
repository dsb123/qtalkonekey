# swift

一个文件服务器

## docker image

docker-qunar.repo.corp.qunar.com/morrisjobke/docker-swift-onlyone

## 操作说明

```
进入container
sudo docker exec -it swift /bin/bash

停止该服务
sudo docker stop swift

启动该服务
sudo docker start swift

重启该服务
sudo docker restart swift

删除该container
sudo docker rm -f swift
```

## swift 提供的端口

8080
